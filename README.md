# Deep Learning Project

## Deskripsi
Project ini merupakan implementasi metode Deep Learning untuk menyelesaikan permasalahan klasifikasi/prediksi menggunakan Python dan Google Colab.

## Tujuan
- Mempelajari implementasi deep learning
- Melatih model menggunakan dataset
- Mengevaluasi performa model
